# Computer Vision lab

Este repositório reúne implementações desenvolvidas durante a trilha de Visão Computacional, abordando técnicas clássicas e modernas para análise de imagens.
Usei um dataset próprio de conchas e pedras.

## Conteúdo

### 1. Detecção e Extração de Características
- Identificação de pontos de interesse em imagens.
- Extração de descritores locais.
- Visualização das características detectadas.

### 2. Correspondência de Características
- Comparação entre duas imagens.
- Associação de descritores.
- Visualização dos matches encontrados.

## Tecnologias

- Python
- OpenCV
- NumPy
- Matplotlib
- PyTorch
- TorchVision

## Competências desenvolvidas

- Coleta de imagens próprias
- Processamento de imagens
- Extração de características
- Correspondência entre imagens
